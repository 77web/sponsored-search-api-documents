# FeedItemService
FeedItemService is to get, add, upgrade, or delete the FeedItem information.
#### WSDL
| environment | url |
|---|---|
| production  | https://ss.yahooapis.jp/services/V5.1/FeedItemService?wsdl|
| sandbox  | https://sandbox.ss.yahooapis.jp/services/V5.1/FeedItemService?wsdl|
#### Namespace
http://ss.yahooapis.jp/V5
#### Overview
Get, add, upgrade, or delete the FeedItem information
#### Operation
Describes operation provided by FeedItemService.
