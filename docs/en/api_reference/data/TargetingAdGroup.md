# TargetingAdGroup
TargetingAdGroup contains ad group setting for Data auto insertion.
### Service
+ [FeedItemService](../services/FeedItemService.md)

| Field | Data Type | maxOccurs | minOccurs | response | add | set | remove | Description | 
|---|---|---|---|---|---|---|---|---|
| targetingAdGroupId| long| 1| 0| yes| Ignore(AD_CUSTOMIZER: Optional)| Ignore(AD_CUSTOMIZER: Optional)| Ignore| Targeting ad group ID.<br>To deactive the setting, insert "0".<br>* There will be no setting, if "0" is set in ADD request. |
