# ReportZip (enum)
ReportZipは、定義されたダウンロードレポートの圧縮の有無を表します。
### Service
+ [ReportDefinitionService](../services/ReportDefinitionService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| OFF| string| ダウンロードレポートは、圧縮されません。（初期値） |
| ON| string| ダウンロードレポートは、ZIP型式で圧縮されます。 |
