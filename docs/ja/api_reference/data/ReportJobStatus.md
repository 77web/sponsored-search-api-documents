# ReportJobStatus (enum)
ReportJobStatusは、定義されたレポートの処理状況です。
### Service
+ [ReportService](../services/ReportService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| COMPLETED| string| 正常に終了しました。 |
| IN_PROGRESS| string| レポートは作成中です。 |
| FAILED| string| 処理が失敗しました。 |
