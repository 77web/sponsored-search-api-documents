# IsDateSpecificRule (enum)
IsDateSpecificRuleは、期限付きルールか定義します。

### Service
+ [RetargetingListService](../services/RetargetingListService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| FALSE| string| 期限ありません。|
| TRUE| string| 期限付きです。|
