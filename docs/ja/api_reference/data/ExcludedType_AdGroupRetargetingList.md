# ExcludedType (enum)
ExcludedTypeは、ユーザー設定の配信ステータスです。

### Service

| 値 | データ型 | 説明 | 
|---|---|---|
| INCLUDED| string| 包含です。 |
| EXCLUDED| string| 除外です。 |
