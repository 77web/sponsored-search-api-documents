# IncludeSubUrlsBool (enum)
IncludeSubUrlsBoolは、対象URLの下位URLを含めるか指定します。
### Service
+ [TargetingIdeaService](../services/TargetingIdeaService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| TRUE| string| 下位URLを含めるときに指定します。 |
| FALSE| string| 下位URLを含めないときに指定します。 |
