# GeoExcludedType (enum)
GeoExcludedTypeは、地域ターゲティング設定での地域の包含または除外を示します。
### Service
+ [CampaignTargetService](../services/CampaignTargetService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| INCLUDED| string| その地域を含みます。 |
| EXCLUDED| string| その地域は除外されます。 |
