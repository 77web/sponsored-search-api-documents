# IncludeUnset (enum)
IncludeUnsetは、日時が設定されていないデータも含めるか指定します。
### Service
+ [CustomerSyncService](../services/CustomerSyncService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| INCLUDED| string| 対象のデータの日時が設定されていないデータを含みます。 |
| EXCLUDE| string| 対象のデータの日時が設定されていないデータを除外します。 |
