# AdGroupAdService
AdGroupAdServiceでは、広告に関する情報の取得および追加・更新・削除を行います。
#### WSDL
| environment | url |
|---|---|
| production  | https://ss.yahooapis.jp/services/V5.1/AdGroupAdService?wsdl|
| sandbox  | https://sandbox.ss.yahooapis.jp/services/V5.1/AdGroupAdService?wsdl|

#### Namespace
http://ss.yahooapis.jp/V5
#### サービス概要
広告に関する情報の取得および追加・更新・削除を行います。　
#### 操作
AdGroupAdServiceで提供される操作を説明します。
